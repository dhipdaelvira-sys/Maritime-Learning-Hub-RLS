Gerakan Mekanis

Semua sistem otomasi kapal dimulai dari sinyal listrik!

A. Gambaran Umum

Apa itu sistem pneumatik? Sistem yang menggunakan udara bertekanan untuk menggerakkan aktuator (silinder, motor).

Mengapa digunakan di kapal? Aman (tidak percikan api), tahan korosi, ringan, dan mudah dikendalikan.

Perbedaan dengan hidrolik: Pneumatik pakai udara (kompresibel), hidrolik pakai oli (tidak kompresibel). Pneumatik lebih aman di area berisiko ledakan.

C. Diagram Alir Sistem

Sumber: Wikimedia Commons

D. Prinsip Kerja (Langkah demi Langkah)

  1. Tombol ditekan → sinyal listrik ON
  2. Kompresor menyala → mengisi tangki udara
  3. Tekanan udara tersimpan di tangki
  4. Solenoid valve aktif → membuka aliran udara
  5. Udara masuk ke silinder → piston bergerak
  6. Udara bekas keluar melalui exhaust

Catatan: Tegangan kontrol umum: 24VDC. Beberapa sistem lama: 220VAC. Kontrol berasal dari panel listrik / PLC / relay.

E. Aplikasi di Kapal

  • Katup kontrol bahan bakar
  • Pintu otomatis engine room
  • Aktuator untuk sistem pemadam kebakaran
  • Alat bantu mesin utama

Sistem Sensor di Kapal

A. Konsep Sensor

Sensor adalah "indra" sistem kontrol. Mengubah kondisi fisik (tekanan, suhu, posisi) menjadi sinyal listrik.

Analog: Output kontinu (misal: 4–20 mA untuk tekanan).

Digital: Output ON/OFF (misal: proximity switch).

B. Jenis-Jenis Sensor

C. Diagram Alir Sinyal

D. Tegangan & Sinyal

  • Tegangan kontrol: 24VDC (standar industri maritim)
  • Sinyal analog: 4–20 mA (lebih tahan noise daripada 0–10V)
  • Sinyal digital: ON = 24V, OFF = 0V

Sistem Starter Motor Listrik di Kapal

1. Gambaran Umum

Starter motor adalah sistem pengasutan motor listrik agar arus & torsi aman.

Digunakan pada pompa, blower, compressor, conveyor kapal.

"Starter adalah jembatan antara sistem listrik dan sistem mekanik."

2. Jenis Starter Motor

A. DOL (Direct On Line Starter)

  • Motor dihubungkan langsung ke sumber
  • Arus start besar
  • Rangkaian sederhana
  • Digunakan untuk motor kecil–menengah

B. Star–Delta Starter

  • Start awal STAR → arus kecil
  • Running DELTA → torsi penuh
  • Digunakan untuk motor besar

3. Diagram Alur Sistem

Power Supply →
MCB →
Contactor →
Overload Relay →
Motor

Sistem kontrol selalu 24VDC, sistem daya bisa 380–440VAC

3A. Pemisahan Sistem DAYA & KONTROL

1. Rangkaian DAYA (Power Circuit)

  • Sumber 3 phase: R – S – T (atau L1 – L2 – L3)
  • Masuk ke MCB / MCCB
  • Masuk ke Contactor (terminal L1 L2 L3)
  • Keluar ke Overload Relay
  • Menuju Motor (terminal U – V – W)

2. Rangkaian KONTROL (Control Circuit)

  • Supply kontrol: 24VDC atau 220VAC
  • Push Button STOP (NC)
  • Push Button START (NO)
  • Coil Contactor (A1 – A2)
  • Kontak bantu NO untuk self-holding
  • Kontak NC Overload (95–96)

3B. Urutan Kerja DOL Starter (Step by Step)

  1. Tombol STOP dalam kondisi normal (NC tertutup)
  2. Tombol START ditekan
  3. Arus mengalir ke coil contactor (A1–A2)
  4. Contactor menarik → kontak utama menutup
  5. Tegangan 3 phase masuk ke motor (U–V–W)
  6. Motor berputar
  7. Kontak bantu NO menahan arus (self-hold)
  8. Jika overload terjadi → kontak 95–96 terbuka → motor mati

3C. Logika Star–Delta Starter

  • Motor harus bertipe 6 terminal (U1 V1 W1 – U2 V2 W2)
  • Start awal menggunakan hubungan STAR
  • Timer aktif menghitung waktu (3–10 detik)
  • Setelah waktu tercapai:
    • Contactor STAR OFF
    • Contactor DELTA ON
  • Interlock wajib agar STAR dan DELTA tidak aktif bersamaan

4. Galeri Komponen

5. Perbandingan DOL vs Star–Delta

DOL:

  • Arus start besar
  • Rangkaian sederhana
  • Murah

Star–Delta:

  • Arus start kecil
  • Rangkaian lebih kompleks
  • Motor lebih awet

6. Aplikasi di Kapal

  • Pompa ballast
  • Pompa pendingin mesin
  • Fan ventilasi engine room
  • Compressor udara

7. Safety & Interlock

  • Overload relay WAJIB aktif
  • Interlock contactor STAR–DELTA
  • LOTO sebelum servis
  • Jangan start motor tanpa beban dicek

8. Kesalahan Umum di Bengkel

  • Salah urutan phase → motor berputar terbalik
  • Overload disetel terlalu besar → motor gosong
  • Kontak bantu tidak dipasang → motor mati saat tombol dilepas
  • STAR & DELTA aktif bersamaan → short circuit
  • Kontrol dan daya disatukan → bahaya fatal
